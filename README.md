# EPUB Translator & Alice Skills 📚 <=> 🌐 <=> 🗣️

Веб-приложение для перевода книг в формате EPUB с использованием Google Gemini API, а также бэкенд для навыков Яндекс.Алисы.

**Основные функции:**

*   **Перевод EPUB:** Загрузка `.epub`, выбор языка/модели, перевод по главам или целиком, пользовательские инструкции для перевода (`prompt_ext`), скачивание результата в `.txt` или `.epub`.
*   **Навык Алисы "Новости BBC":** Озвучивает последние заголовки новостей с BBC News, переведенные на русский язык (использует кеширование).
*   **Навык Алисы "Умный Помощник":** Позволяет задавать вопросы Алисе и получать ответы, сгенерированные Google Gemini (с обработкой ожидания).

## ✨ Возможности (EPUB Переводчик)

*   **Загрузка EPUB:** Простой интерфейс для загрузки файлов `.epub`.
*   **Персистентное Хранение:** Используется база данных SQLite (`epub_translator.db`) для хранения информации о книгах, секциях, их статусах и пользовательских инструкциях (`prompt_ext`).
*   **Автоматический парсинг:** Извлечение структуры книги и оглавления (TOC).
*   **Перевод оглавления:** Автоматический перевод названий глав в TOC при загрузке книги.
*   **Выбор модели и языка:** Возможность выбрать целевой язык перевода и модель Google Gemini. Выбор сохраняется в сессии браузера.
*   **Пользовательские инструкции (`prompt_ext`):** Возможность задать текстовые инструкции для всей книги (правила перевода имен, терминов), которые будут учтены моделью при переводе. Сохраняются в БД.
*   **Фоновый перевод:** Перевод секций выполняется в фоновом режиме (`ThreadPoolExecutor`), не блокируя интерфейс.
*   **Обновление статуса:** Отображение статуса перевода для каждой секции (не переведено, в процессе, переведено (с моделью), ошибка, пусто) с обновлением в реальном времени (через поллинг).
*   **Просмотр перевода:** Отображение переведенного текста выбранной секции в браузере.
*   **Кэширование:** Переведенные секции кэшируются на диске (`.epub_cache/`).
*   **Скачивание результатов:** Скачивание отдельных секций (`.txt`), всего текста (`.txt`) или готовой книги (`.epub`).
*   **Управление книгами:** Удаление загруженных книг, их записей в БД и кэша.

## ✨ Возможности (Навыки Алисы)

*   **Новости BBC (`/alice`):**
    *   Получает последние заголовки с BBC News через NewsAPI.
    *   Переводит заголовки на русский с помощью Gemini.
    *   Кеширует переведенные заголовки в памяти для быстрого ответа.
    *   Озвучивает несколько последних заголовков по запросу к Алисе.
*   **Умный Помощник (`/alice/smart`):**
    *   Принимает вопрос пользователя от Алисы.
    *   Отправляет запрос к Google Gemini (например, `gemini-1.5-pro`) для генерации ответа.
    *   Обрабатывает возможное долгое ожидание ответа Gemini с помощью промежуточного ответа Алисе ("Пока думаю...").
    *   Возвращает сгенерированный и обрезанный по длине ответ пользователю.

## 🚀 Технологический стек

*   **Бэкенд:** Python 3, Flask
*   **База данных:** SQLite 3
*   **API Перевода/Генерации:** Google Gemini API (`google-generativeai`)
*   **API Новостей:** NewsAPI (`requests`)
*   **Планировщик (для новостей):** `APScheduler` (Опционально, если используется фоновое обновление кеша новостей)
*   **Парсинг/Генерация EPUB:** `ebooklib`
*   **Фронтенд (EPUB):** HTML, CSS, JavaScript (ванильный)
*   **Асинхронность:** `concurrent.futures.ThreadPoolExecutor`

## 📋 Предварительные требования

*   Python 3 (рекомендуется 3.9+)
*   `pip` (менеджер пакетов Python)
*   **Google API Key:** Активный ключ с доступом к Google Gemini API. [Как получить ключ](https://aistudio.google.com/app/apikey)
*   **NewsAPI Key:** Активный ключ для доступа к NewsAPI (для навыка новостей BBC). [Как получить ключ](https://newsapi.org/)

## ⚙️ Установка и настройка

1.  **Клонируйте репозиторий.**
2.  **Создайте и активируйте виртуальное окружение.**
3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Убедитесь, что `requests` и `APScheduler` (если используется) добавлены в `requirements.txt`)*
4.  **Настройте API ключи:**
    *   **Рекомендуемый способ:** Использовать файл `.env` в корне проекта и библиотеку `python-dotenv`.
        *   Установите: `pip install python-dotenv`
        *   Создайте файл `.env`:
            ```env
            GOOGLE_API_KEY=ВАШ_GOOGLE_КЛЮЧ
            NEWSAPI_KEY=ВАШ_NEWSAPI_КЛЮЧ
            ```
        *   **Добавьте `.env` в `.gitignore`!**
        *   Добавьте в начало `app.py` (или WSGI-файла при развертывании):
            ```python
            from dotenv import load_dotenv
            load_dotenv()
            ```
    *   **Альтернатива:** Установить переменные окружения `GOOGLE_API_KEY` и `NEWSAPI_KEY` в вашей системе или терминале перед запуском.

## ▶️ Запуск приложения

1.  Убедитесь, что `venv` активно и ключи API настроены.
2.  Запустите Flask приложение:
    ```bash
    python app.py
    ```
    *(Будет создан файл `epub_translator.db` при первом запуске)*.
3.  **Для EPUB Переводчика:** Откройте `http://127.0.0.1:5000` (или ваш IP/порт).
4.  **Для Навыков Алисы:** Веб-сервер должен быть доступен из интернета. Настройте соответствующие вебхуки в консоли Яндекс.Диалогов:
    *   Новости BBC: `https://<ВАШ_ПУБЛИЧНЫЙ_URL>/alice`
    *   Умный Помощник: `https://<ВАШ_ПУБЛИЧНЫЙ_URL>/alice/smart`
    *(Для локального теста используйте ngrok)*.

## 📖 Использование (EPUB Переводчик)

1.  **Загрузка:** Загрузите `.epub`, выберите язык.
2.  **Просмотр:** Откройте книгу.
3.  **Настройка:** Выберите язык/модель.
4.  **Инструкции:** Введите правила перевода в поле "Дополнительные инструкции", нажмите "Сохранить".
5.  **Перевод:** Клик по главе или кнопка "Перевести все". Инструкции будут учтены.
6.  **Скачивание:** Кнопки `💾`, "Скачать TXT", "Скачать EPUB" (последние активны после завершения).
7.  **Удаление:** На главной странице.

## 📖 Использование (Навыки Алисы)

1.  **Новости BBC:** Скажите Алисе активационное имя (например, "Запусти новости БиБиСи"). Навык озвучит последние переведенные заголовки. (Кеш обновляется лениво при первом запросе или по расписанию, если настроен APScheduler).
2.  **Умный Помощник:** Скажите активационное имя (например, "Запусти умного помощника") и задайте вопрос. Навык ответит "Сейчас подумаю...", обработает запрос с помощью Gemini в фоне и выдаст ответ, когда вы скажете что-нибудь еще (например, "угу", "что там?").

## ⚠️ Ограничения и Предупреждения

*   **Производительность:** SQLite может тормозить при больших объемах. Фоновые задачи могут нагружать сервер.
*   **Стоимость API:** Google Gemini и NewsAPI могут быть платными.
*   **Таймауты Алисы:** Навыки должны отвечать быстро (~3-4 сек). Реализованы механизмы кеширования (новости) и ожидания (умный помощник).
*   **Качество:** Зависит от моделей и промптов.
*   **Безопасность:** Предназначено для локального использования или доверенной среды. Требует мер безопасности при публичном развертывании.
*   **Зависимость от `translation_module`:** `alice_handler` для новостей импортирует `translate_text`. Если этот модуль не будет доступен или изменится, перевод новостей сломается.
